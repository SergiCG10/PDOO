.discardElement(usesLeft) ⇒ Object

Funcion discardElement. Muestra true si se debe de descartar o no un objeto. Se calcula si se decarta de forma que : Si tiene el número máximo de usos, no se descartará, Si tiene 0 se descartará de forma asegurada En cualquier otro caso, la probabilidad de que se descarte será proporcionalmente inversa al número de usos que le quedan respecto al número máximo de usos.

Parameters:

  • usesLeft —

    Número de usos restantes

Returns:

  • True si se descarta, false si no

def self.discardElement(usesLeft)
	discard = true;
	if (usesLeft == @@MAX_USES)
	    discard = false;
	elsif(usesLeft != 0)
	    discard = @@generator.rand < 1 - usesLeft.to_f/@@MAX_USES;
	end
	return discard; 
end
